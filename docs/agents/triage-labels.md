# Triage 标签

各 skill 以五个规范 triage 角色表述状态。本文件把这些角色映射到本仓库 issue tracker 实际使用的标签字符串。

| skill 中的角色 | 本仓库标签 | 含义 |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 维护者需要评估此 issue |
| `needs-info` | `needs-info` | 等待报告人补充信息 |
| `ready-for-agent` | `ready-for-agent` | 规格完整,可由 AFK agent 领取 |
| `ready-for-human` | `ready-for-human` | 需要人工实现 |
| `wontfix` | `wontfix` | 不会处理 |

当 skill 提到某个角色(例如"打上 AFK-ready triage 标签")时,使用上表对应的标签字符串。

如需更换词汇,直接编辑右列即可。
